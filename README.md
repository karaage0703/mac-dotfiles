# mac-dotfiles
private dotfiles for mac

# Install
Execute following commands  
(Please install `brew` before execute commands)

```sh
$ cd
$ git clone https://github.com/karaage0703/mac-dotfiles.git
$ cd ~/mac-dotfiles
$ ./homebrew.sh
$ ./sync.sh
```

# Update
```sh
$ cd ~/mac-dotfiles
$ ./sync.sh
```
