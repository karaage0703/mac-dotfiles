# mac-dotfiles
private dotfiles for mac

# Install
Execute following commands

```sh
$ cd
$ git clone https://github.com/karaage0703/mac-dotfiles.git
$ cd ~/mac-dotfiles
$ ./sync.sh
```

# Update
```sh
$ cd ~/mac-dotfiles
$ ./sync.sh
```
